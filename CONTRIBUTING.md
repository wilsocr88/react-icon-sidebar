# Contributing

Thanks for your interest in this React component. Contributions are welcome.

Start with existing [issues](https://github.com/wilsocr88/react-icon-sidebar/issues). Feature ideas/requests are welcome.

## Before You Start

- Keep changes small and focused.
- Match the existing JSX and JavaScript style in the repo.
- Avoid introducing new dependencies unless they are clearly needed.

## Local Checks

Run the available checks before opening a pull request:

```bash
npm test
npm run build
```

## Pull Requests

- Describe what changed and why.
- Mention any behavior changes or edge cases.
- Include tests when you add or modify behavior.
- Keep the README up to date if the public API changes.

## Reporting Issues

- Include the command or code path that failed.
- Share the expected and actual behavior.
- Add a minimal reproduction when possible.
