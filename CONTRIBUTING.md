# CONTRIBUTING

Thanks for your interest in this React component. Contributions are welcome.

Feel free to [submit a pull request](https://github.com/wilsocr88/react-icon-sidebar/pulls)
or, if you find a bug, [an issue](https://github.com/wilsocr88/react-icon-sidebar/issues)
(make sure it's a bug in the component, and not in your code or toolset!).
